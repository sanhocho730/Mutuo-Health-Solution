Autoscribe High Level Design

Components

- (PARSER) Parse Forms into machine friendly representation.
    - output jpg from pdf given as input
    - encode jpg into base64 for OpenAI requests
    - set prompt and send OpenAI requests
- (FILLER) Fill in the machine friendly representation of forms using information extracted from natural language texts.
- (OUTPUT) Change the filled up machine friendly representation of forms back into human readable forms like PDF.

Characteristic Requirements

1. machine friendly representation of forms



Functional Requirements

- GLOBAL_REQ_1
    input:
        - unfilled forms
        - natural language texts that contain information that can fill in forms
    output:
        - filled in forms
